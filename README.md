DSH Institute of Technology  Academic Portal
A high-performance, fully functional web ecosystem developed for DSH Institute of Technology (DIT). This project is a comprehensive solution for modern campus management, handling everything from student inquiries and feedback to complex enrollment workflows and accommodation logistics.

The Vision
This portal was designed to move beyond the static nature of traditional academic websites. It is a 100% functional tool built from the ground up to streamline the student journey. Whether a student is selecting a program, requesting housing, or submitting official enrollment data, the system handles the logic with precision.

Design & Experience
The website features a bespoke visual identity and a cutting-edge user experience:
Vibrant Duo-Tone Aesthetic: A bold, modern color palette that ensures high readability and a professional, energetic feel.
Unique Animations: Custom-coded, modern animations that provide fluid transitions and interactive feedback without compromising performance.
Signature Header & Footer: An "amazing" architectural design for the navigation and footer sections, ensuring global site consistency and ease of access.
Built from Scratch: No templates were used. Every line of CSS, every layout grid, and every interactive element was custom-engineered.

Functional Core
Program Management: Dynamic handling of the college's 4 primary academic tracks.
Enrollment System: A robust portal for both local and international students featuring real-time validation and nationality logic.
Integrated Inquiry Hub: Centralized forms for general inquiries, accommodation requests, and administrative feedback.
Document Integration: Seamless handling of digital PDFs for registration and health declarations.
